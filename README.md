## Getting Started

1. Clone the repository:
    ```bash
    git clone https://github.com/your-username/NestLearn.git
    ```
2. Navigate to the desired project directory:
    ```bash
    cd NestLearn/Project1
    ```
3. Install dependencies:
    ```bash
    npm install
    ```
4. Run the project:
    ```bash
    npm run start
    ```
